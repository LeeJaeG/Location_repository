# echo-e_Location

## 설치 방법

1. 이 저장소를 클론합니다.
   ```
   git clone https://github.com/LeeJaeG/echo-e_Location.git
   ```

2. 필요한 패키지를 설치합니다.
   ```
   pip install -r requirements.txt
   ```

## 사용 방법

1. 애플리케이션을 실행합니다.
   ```
   python main.py
   ```

2. 브라우저에서 `http://localhost:8000`으로 접속합니다.
